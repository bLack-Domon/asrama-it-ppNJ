Bermain HTML Dasar

  1. Pengertian HTML

    HTML (HyperText Markup Language) adalah bahasa markah yang digunakan untuk membuat dan mendesain struktur halaman web. Ini adalah bahasa inti yang digunakan untuk membangun suatu situs web. HTML memungkinkan penataan dan penampilan elemen-elemen konten, seperti teks, gambar, tautan, formulir, video, dan lainnya pada halaman web.

  2. Kerangka HTML

    • Doctype : Ini adalah tag penjelas tentang versi HTML
    • HTML : tag ini menjelaskan bahwa file ini berformat html
    • Head : berisikan meta, link css atau tautan lain, dan judul website
    • Body : berisikan kerangka atau desain untuk halaman website yang dibuat.
